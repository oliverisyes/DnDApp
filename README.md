# DnDApp
> [!IMPORTANT]
> This project is still in active development, major features are still missing/non-functional.

A Windows desktop application to replace existing ttrpg character builders.
Improvements on existing apps include: runs completely locally without needing a resource intensive web browser to be accessed, doesn't sell your data, there are no forced updates and character data will always be available even if the app is rendered inaccessable for some reason. 

This app was built with Dungeons and Dragons in mind however if your preferred ttrpg uses similar character sheets, it may also be compatible.
